- HTML5
- CSS3
- JavaScript (DOM Manipulation)
- Local Storage

- Add Student Records
- Edit Student Records
- Delete Student Records
- Search Students
- Form Validation
- Dynamic Scrollbar
- Responsive Design
- Local Storage Support
- Student Counter
- Toast Notifications
